# i3wm-config
Configuration file for my i3-gaps environment

- Distro: Arch Linux <br />
- wm: i3-gaps <br />
- Terminal: Alacritty <br />
- Alacritty-theme: Doomicide Darkocean <br />
- Background manager: Feh <br />
